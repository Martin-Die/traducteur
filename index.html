Comment utiliser le traducteur

  • Conversion Texte vers Brainfuck :
    • Écrivez votre texte dans la zone "Texte normal"
    • Le code Brainfuck optimisé sera généré automatiquement
    • Utilisez le bouton "Copier le code" pour copier le résultat
  • Conversion Brainfuck vers Texte :
    • Collez votre code Brainfuck dans la zone "Code Brainfuck"
    • Le texte décodé apparaîtra automatiquement
    • Utilisez le bouton "Copier le texte" pour copier le résultat
  • Traducteur d'Opérations :
    • Entrez une opération mathématique simple (+, -, *, /)
    • Le code Brainfuck correspondant sera généré
    • Vous pouvez aussi convertir du Brainfuck en opération
  • Optimisations :
    • Le code généré utilise des boucles pour réduire la taille
    • Exemple : le caractère 'A' (65) devient +++++[>++++++++++<-]>+. au lieu de 65 '+'
    • Les caractères répétés sont optimisés automatiquement
  • Fonctionnalités :
    • Conversion bidirectionnelle en temps réel
    • Support complet des caractères ASCII (0-255)
    • Interface responsive adaptée aux mobiles
